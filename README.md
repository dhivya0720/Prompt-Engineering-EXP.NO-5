int factorial(int n) {
int i, fact = 1;
for(i = 1; i <= n; i++) {
fact *= i;
}
return fact;
}

int num;
printf("Enter a number: ");

Summary of Findings
Prompt clarity has a direct impact on response quality.
Basic prompts improve organization, relevance, and completeness.
Adding context, constraints, and expected format leads to better outputs.
Naïve prompts are suitable for quick information retrieval but may generate generic responses.
For academic, technical, and professional tasks, structured prompts provide significantly better results.
Insights for Writing Effective Prompts
Clearly state the topic and objective.
Specify the desired length or format.
Include important points that must be covered.
Mention the target audience when relevant.
Request examples, explanations, or step-by-step details if needed.
